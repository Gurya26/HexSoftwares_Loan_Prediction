# Loan Eligibility Prediction

## Project Overview
This project predicts whether a loan application will be approved or not using Machine Learning.

It includes:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training (Logistic Regression)
- Model Evaluation


## Dataset
The dataset contains information about applicants such as:
- Gender
- Marital Status
- Income
- Loan Amount
- Credit History
- Property Area

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib & Seaborn
- Scikit-learn

## Exploratory Data Analysis
- Missing values handled using mean & mode
- Data types analyzed
- Statistical summary generated

## Visualizations
- Loan Approval Distribution
- Feature Correlation Heatmap
- Feature Impact Analysis

## Model Used
- Logistic Regression

## Model Performance
- Accuracy: **77%**
- Confusion Matrix
- Classification Report

## Key Insights
- Credit History is the most important factor
- Most loans are approved in dataset
- Model performs better on approved cases
